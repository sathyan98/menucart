# Features Implemented
  1.) Cart Functionality
  2.) Adding and Removing items from Cart
  3.) Filter Functionality
  4.) Responsive Design
  5.) Dynamic Routing
  6.) Generating a Dynamic card

# Javascript and React Features
   The application has deigned using these functionalities - React Router, Dynamic Routing, Redux toolkit, Promises etc,

# Note :
